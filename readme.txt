Maintained toolsets: msvc, msvc-x64, macosx-10.7, linux, linux-x64, iphone-device-armv7, iphone-device-arm64, iphone-simulator-x86, iphone-simulator-x86-64, android-armv7, android-arm64 

Supported MSVS versions - 2015 or newer

ubuntu packages needed for compilation: libasound2-dev libx11-dev libegl1-mesa-dev libgles2-mesa-dev
ubuntu packages needed for 32bit compilation: gcc-multilib g++-multilib libgl1-mesa-glx:i386 libx11-dev:i386 libgles2-mesa-dev:i386

For android builds following sdk packages are required: sdk tools (obsolete), ndk, sdk platform 28, build tools 28.0.3
